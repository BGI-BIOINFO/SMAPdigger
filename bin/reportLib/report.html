Introduction

Typically, RRBS samples are generated by digesting genomic DNA with the restriction endonuclease MspI. This is followed by insert-size select (50-220bp), ending repair, A-tailing, adapter ligation and finally bisulfite conversion. Often, the library is also size-selected for fragments between 40 and 220bp in length, but we just use 50-220bp fragments in our work flow. This fragment size has been shown to be plentiful in the sample and yield information on the vast majority of CpG islands (CGIs) in the human or mouse genome. File Format

Fastq file format

A FASTQ file normally uses four lines per sequence. Line 1 begins with a '@' character and is followed by a sequence identifier and an optional description (like a FASTA title line). Line 2 is the raw sequence letters. Line 3 begins with a '+' character and is optionally followed by the same sequence identifier (and any description) again. Line 4 encodes the quality values for the sequence in Line 2, and must contain the same number of symbols as letters in the sequence. The base's quality is equal to ASCII value in line that start from 33 (Sanger FASTQ-like format) / 64 (Illumina 1.3+ FASTQ-like format). Starting from the Illumina GA Pipeline v1.7, the range of base quality values is from 2 to 41 (66 - 105).
